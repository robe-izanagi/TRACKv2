Project Overview:
TRACK Version 2 is a customizable calendar scheduler developed specifically for the Polytechnic University of the Philippines (PUP) – Sto. Tomas. 
It is designed as a hybrid mobile-web application to eliminate scheduling conflicts, improve time management, and optimize the allocation of university resources (venues, equipment) for officials, faculty, and staff.

Core Features:

• Cross-Platform Setup: 
A React JS Web App (for Admins) and a React JS Mobile App (wrapped with Capacitor) for universal access across iOS and Android.
• Three Calendar Types: 
Personal, Departmental, and Campus-wide events with strict role-based visibility.
• Smart Conflict Resolution: 
Uses a hierarchy system (Local → Regional → National → International) and event types (Campus → Dept → Private) to recommend which event to prioritize during overlaps.
• AI-Driven Assistance: 
Provides optimized schedule suggestions based on attendee availability and venue booking history.
• Task & Venue Management: 
Includes task checklists, progress tracking, and a staff-managed venue allocation module.
• Analytics & Feedback: 
Admin dashboards with usage metrics, predictive conflict forecasts, and a text-mining system that automatically extracts common keywords from user feedback.
• Controlled Onboarding: 
Uses a unique "Account Code" system generated by admins to ensure only authorized PUP personnel can register.


Tech Stack (Updated):

• Frontend (Web & Mobile): React JS (with Capacitor for mobile deployment)
• Backend: Express.js + Node.js
• Database: MySQL (Cloud-hosted)
• Methodology: Agile
